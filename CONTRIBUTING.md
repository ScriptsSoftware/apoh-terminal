APOH CONTRIBUTING RULES

📌 Reglas obligatorias

Los scripts deben ser archivos .sh

Deben funcionar en Termux sin modificaciones externas

No se permiten comandos destructivos (ej: rm -rf /, mkfs, etc.)

El código debe ser limpio, legible y comentado cuando sea posible

Cada script debe incluir nombre y propósito claro

Se debe incluir el código fuente completo del script

Gracias :)
